Le projet est une appplication de gestion de comptes bancaires par les employés de cette banque.

Vous trouvez le fichier contenant le code source de l'application et un fichier "database" qui contient le script au language Mysql. 

Les classes Bank() et Authentification(), sont les  classes qui assurent  la connection  avec la   base de donnée en utilisant JDBC.

afin de se connecter à la base de données, pensez a changer le url, username et le password dans la classe bank() et Authentification().
